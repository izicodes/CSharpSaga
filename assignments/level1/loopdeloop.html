Loop De Loop!

Status:

Introduction

Control structures and loops are fundamental to any programming language. In this assignment, you'll be creating a program that uses these constructs to print a series of numbers and their squares.

Example Output

Having examples can really help on giving you an ideas of what your solution could output. If stuck, try achieving this example output:

Please enter a number: 5
1 square is 1
2 square is 4
3 square is 9
4 square is 16
5 square is 25

Instructions

Here are some instructions that you could follow to complete the assignment:

  1. Write a program that asks the user to input a number.
  2. Using a loop, print all numbers from 1 up to and including the input number, along with the square of each number.
  3. Use an appropriate control structure to ensure that the input number is positive.
  4. Think about the loop control structure that would be most appropriate for this task.

using System;

class Program
{
    static void Main(string[] args)
    {
        Console.Write("Please enter a number: ");
        int number = int.Parse(Console.ReadLine());

        // Check if the number is positive
        while (number <= 0)
        {
            Console.WriteLine("Invalid input. Please enter a positive number: ");
            number = int.Parse(Console.ReadLine());
        }

        // Print numbers and their squares
        for (int i = 1; i <= number; i++)
        {
            int square = i * i;
            Console.WriteLine($"{i} square is {square}");
        }
    }
 }
