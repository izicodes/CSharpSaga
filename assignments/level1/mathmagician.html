Math Magician!

Status:

Introduction

This assignment will have you putting C#'s operators to work. You'll be creating a calculator that can perform basic arithmetic operations.

Example Output

Having examples can really help on giving you an ideas of what your solution could output. If stuck, try achieving this example output:

Enter the first number: 10
Enter the second number: 2
Choose operation: +, -, *, / : *
The result of the operation is 20.

Instructions

Here are some instructions that you could follow to complete the assignment:

  1. Write a program that prompts the user to input two numbers.
  2. Ask the user to choose an operation from the following: addition, subtraction, multiplication, or division.
  3. Perform the chosen operation on the two numbers and print the result.
  4. Ensure you handle the case where a user might try to divide by zero.

using System;

class Program
{
    static void Main(string[] args)
    {
        Console.Write("Enter the first number: ");
        double firstNumber = double.Parse(Console.ReadLine());

        Console.Write("Enter the second number: ");
        double secondNumber = double.Parse(Console.ReadLine());

        Console.Write("Choose operation: +, -, *, / : ");
        char operation = char.Parse(Console.ReadLine());

        double result = 0;

        switch (operation)
        {
            case '+':
                result = firstNumber + secondNumber;
                break;
            case '-':
                result = firstNumber - secondNumber;
                break;
            case '*':
                result = firstNumber * secondNumber;
                break;
            case '/':
                if (secondNumber != 0)
                {
                    result = firstNumber / secondNumber;
                }
                else
                {
                    Console.WriteLine("Error: Division by zero is not allowed.");
                    return;
                }
                break;
            default:
                Console.WriteLine("Error: Invalid operation selected.");
                return;
        }

        Console.WriteLine("The result of the operation is " + result + ".");
    }
}

*It's important to remember, this solution matches the example solution given above. Also, there are many ways to achieve the same output!
